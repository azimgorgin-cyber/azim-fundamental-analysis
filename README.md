# دستیار تحلیل بنیادی عظیم

یک پلتفرم فارسی، ساده و قابل توسعه برای تحلیل بنیادی حرفه‌ای سهام.

## نسخه نخست

- نمای کلی وضعیت بنیادی شرکت
- نسبت‌ها و شاخص‌های مالی مهم
- نمایش روند تاریخی صورت‌های مالی
- نمودار پنج‌ساله فروش و سود خالص با جزئیات هنگام نگه‌داشتن موس
- مقایسه فصلی فروش و سود خالص ۱۴۰۳ و ۱۴۰۴
- قیمت، مشخصات نماد و سهامداران عمده از TSETMC
- سرمایه ثبت‌شده و صورت‌های مالی از کدال
- سناریوهای ساده ارزش‌گذاری
- تز سرمایه‌گذاری و عوامل ابطال تحلیل

داده‌های تاریخی فعلی مربوط به نماد دزاگرس و برگرفته از گزارش‌های رسمی کدال‌اند. قیمت و سهامداران عمده از سرویس رسمی TSETMC دریافت می‌شوند. بخش ارزش‌گذاری هنوز فقط مضارب بازار را نمایش می‌دهد و توصیه خرید یا فروش نیست.

## اجرای محلی

پیش‌نیاز: Node.js نسخه 22.13 یا جدیدتر و pnpm.

```bash
pnpm install
pnpm dev
```

## ساخت نسخه نهایی

```bash
pnpm build
```
