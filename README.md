# دستیار تحلیل بنیادی عظیم

یک پلتفرم فارسی، ساده و قابل توسعه برای تحلیل بنیادی حرفه‌ای سهام.

## نسخه نخست

- نمای کلی وضعیت بنیادی شرکت
- نسبت‌ها و شاخص‌های مالی مهم
- نمایش روند تاریخی صورت‌های مالی
- سناریوهای ساده ارزش‌گذاری
- تز سرمایه‌گذاری و عوامل ابطال تحلیل

داده‌های فعلی برنامه نمایشی‌اند و برای تصمیم‌گیری سرمایه‌گذاری مناسب نیستند. اتصال به منابع داده واقعی در مراحل بعدی انجام می‌شود.

## اجرای محلی

پیش‌نیاز: Node.js نسخه 22.13 یا جدیدتر و pnpm.

```bash
pnpm install
pnpm dev
```

## ساخت نسخه نهایی

```bash
pnpm build
```
