# دستیار تحلیل بنیادی عظیم

یک پلتفرم فارسی، ساده و قابل توسعه برای تحلیل بنیادی حرفه‌ای سهام.

## نسخه نخست

- نمای کلی وضعیت بنیادی شرکت
- نسبت‌ها و شاخص‌های مالی مهم
- نمایش روند تاریخی صورت‌های مالی
- نمودار پنج‌ساله فروش و سود خالص با جزئیات هنگام نگه‌داشتن موس
- مقایسه فصلی فروش و سود خالص ۱۴۰۳ و ۱۴۰۴
- قیمت، مشخصات نماد و سهامداران عمده از TSETMC
- سرمایه ثبت‌شده و صورت‌های مالی از کدال
- سناریوهای ساده ارزش‌گذاری
- تز سرمایه‌گذاری و عوامل ابطال تحلیل

داده‌های تاریخی فعلی مربوط به نماد دزاگرس و برگرفته از گزارش‌های رسمی کدال‌اند. قیمت و سهامداران عمده از سرویس رسمی TSETMC دریافت می‌شوند. بخش ارزش‌گذاری هنوز فقط مضارب بازار را نمایش می‌دهد و توصیه خرید یا فروش نیست.

## اجرای محلی

پیش‌نیاز: Node.js نسخه 22.13 یا جدیدتر و pnpm.

```bash
pnpm install
pnpm dev
```

## ساخت نسخه نهایی

```bash
pnpm build
```

## ساخت نسخه GitHub Pages

```bash
pnpm build:pages
```

خروجی استاتیک در پوشه `pages-dist` ساخته می‌شود و قیمت و سهامداران را مستقیماً از سرویس عمومی TSETMC دریافت می‌کند.

## به‌روزرسانی داده‌های رسمی

```bash
pnpm data:update
```

این فرمان گزارش‌های فعالیت ماهانه دزاگرس را از کدال و آخرین قیمت و سهامداران را از TSETMC دریافت می‌کند، فایل داده رسمی را به‌روزرسانی می‌کند و کنترل تطبیق فروش ماهانه با صورت مالی سالانه را دوباره محاسبه می‌کند.

مخزن عمومی دارای شاخه `source` است. در محل کار می‌توان پروژه را مستقیماً کلون کرد:

```bash
git clone https://github.com/azimgorgin-cyber/azim-fundamental-analysis.git
cd azim-fundamental-analysis
pnpm install
pnpm dev
```

گردش‌کار GitHub Actions هر روز داده‌های رسمی را کنترل می‌کند و در صورت تغییر، شاخه `source` و نسخه GitHub Pages را به‌روزرسانی می‌کند. اجرای دستی آن نیز از بخش Actions امکان‌پذیر است.
